# Passenger list app

## Tech

- React
- Vite
- TypeScript
- SCSS

## API

Uses: https://fake-json-api.mock.beeceptor.com/users

## Features

- Load initial passengers (5 items)
- Show more (adds 5 more, max 5 loads)
- Add passenger
- Skeleton loaders for initial load & load more
- Smooth scroll to bottom on append

## Scripts

```bash
npm install
npm run dev
npm run build
```
